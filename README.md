# Student-Attendance-Management-System
A web-based Attendance Management System designed to simplify recording, tracking, and managing student attendance. This project helps automate the traditional manual attendance process using a clean UI, secure authentication, and efficient database management.
The attendance shortage reports can be downloaded and will be useful for informing their parents regarding attendance shortage.
Helpful for students and their parents to maintain proper attendance.


Technologies used:
python, html, css, Flask, sql
